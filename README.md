# vba formula parser

A library to parse formulas

## Installation

Import Formulas.bas into your project
Include "Microsoft Scripting Runtime"

## Supported Syntax

- Operators
  - Arithmetic (+, -, \*, /, ^)
  - Comparison (=, <>, <, >, <=, >=)
  - Concatenation (&)
- Function Calls
- Constants
  - Numeric
  - String
  - Logical (TRUE, FALSE)
  - Array constants (e.g., {1,2;3,4}).

## Features

- Formula Formatting
- AST Generation
